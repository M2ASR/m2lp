
"m2char" folder contains a character and acoustic processing framework for multi-languages.
the contennts are: 
./doc: manual files which discribe this tool.
./rc: code-map-files for various languages, used for character transformation from unicode to ascII codes.
./ut: a unit test folder.

This tool accomplish code transformation for various less popular languages whose character-codes are diverse, 
and neccesary to be unified into un-ambiguous codes. Or , we can call it character normalization.

As a part of ASR project this tool can produce acoutic dictionary and acoutic files for various languages.

This tool also include spelling checking for various languages according to fundamental rules like: syllable structure, prouncation rules, ect.
